This is currently in alpha testing and only used by the class of 19SH07. If this thing is good and can be scaled up then we will make this into beta release. Have a nice day:)

Remember those shitty order forms or that crapy excel system that you use to glue everything together? Now with this system it is all integrated into one platform for both the person who is collecting the money as well as the person that has to put in all the orders. As such this should help my class chair person much more to get off his lazy ass. Anyways, head over to the order form section of this website, where there is a well... order form.
